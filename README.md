# Orange-Dashboard

## Data Architecture
Data architecture explains how data is handled, from the moment it is collected to when it is changed, given out, and used. It makes a plan for how data should be stored and how it should move through different storage systems. This is important for doing things with data, such as using it in AI.
 
The data architecture of a dashboard that uses Kibana and Elasticsearch involves the components such as Data Collection in which the data can be in different formats, such as log files, CSV files, or databases. Data Storage in which is used to collect data and then stored it in Elasticsearch, which is a distributed, open-source search and analytics engine. Data Indexing in which Elasticsearch creates a map of the data so that it can be searched and analyzed more efficiently. Data Visualization in which Kibana is used to visualize the data stored in Elasticsearch. Data Analysis in which data can be easily analyzed and queried to uncover insights and trends.
